Conectar Wallet

Conectar tu Wallet

Guía para Conectar tu Billetera a Brutus.finance

Wallet Recomendada: TronLink

TronLink es la wallet más popular y recomendada para interactuar con Brutus.finance y el ecosistema Tron.

Instalación

Extensión de Navegador

  1. Ve a tronscan.org o busca "TronLink" en la Chrome Web Store
  2. Haz clic en "Añadir a Chrome" (o Firefox/Brave)
  3. Espera la confirmación de instalación

Crear una Nueva Wallet

  1. Haz clic en "Crear nueva wallet" en TronLink
  2. Guarda tu frase semilla (12 o 24 palabras) en un lugar seguro
    • ⚠️ NUNCA compartas tu frase semilla con nadie
    • Escríbela en papel y guárdala en un lugar seguro
  3. Confirma la frase semilla en el orden correcto
  4. Establece una contraseña segura para la wallet
  5. ¡Tu wallet está lista!

Importar una Wallet Existente

  1. Selecciona "Importar wallet" en TronLink
  2. Elige el método:
    • Frase semilla (12/24 palabras)
    • Archivo keystore + contraseña
    • Clave privada
  3. Ingresa la información solicitada
  4. Establece una nueva contraseña

Conectar a la dApp de Brutus

Método 1: dApp Web (Escritorio)

  1. Abre dapp.brutus.finance
  2. Haz clic en el botón "Conectar Wallet"
  3. Selecciona TronLink de la lista de wallets disponibles
  4. Autoriza la conexión desde la ventana emergente de TronLink
  5. ¡Conectado! Puedes ver tu dirección y balance

Método 2: Bot de Telegram

  1. Abre el Brutus Energy Bot en Telegram
  2. Inicia el comando /start
  3. El bot te guiará para conectar tu wallet
  4. Escanea el código QR o sigue las instrucciones

Método 3: API (Desarrolladores)

  • Usa las endpoints de la API con tu clave API
  • Autenticación mediante firma de mensajes

Seguridad de tu Wallet

✅ Buenas Prácticas

  • Guarda tu frase semilla en un lugar físico seguro
  • Nunca compartas tu frase semilla ni clave privada
  • Activa la autenticación de dos factores si es posible
  • Verifica siempre las URLs antes de conectar tu wallet
  • Usa una wallet dedicada para DeFi (no almacenes todo en una sola)
  • Revisa los permisos que otorgas a cada dApp

❌ Lo que NUNCA debes hacer

  • Compartir tu frase semilla por email, Telegram o redes sociales
  • Introducir tu frase semilla en cualquier sitio web
  • Hacer clic en enlaces sospechosos
  • Conectar tu wallet a sitios no verificados

Solución de Problemas

Problema Solución
La dApp no detecta TronLink Asegúrate de estar en la red Mainnet de Tron
Error de conexión Refresca la página y vuelve a conectar
No veo mi balance Verifica que estás conectado a la red correcta
Transacción fallida Verifica que tienes suficiente energía y bandwidth
Token no aparece Añádelo manualmente en TronLink
